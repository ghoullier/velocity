# AngularJS Boilerplate

Provide structure and automation to manage AngularJS projects

Based on https://github.com/Hyra/angular-gulp-browserify-livereload-boilerplate

## Dependencies

### CommonJS Modules

- browserify
- gulp-browserify

---

### Web Server and Livereload

- express
- connect-livereload
- gulp-livereload

---

### JS utilities

- gulp-jshint
- gulp-uglify

---

### CSS utilities

- gulp-autoprefixer
- gulp-sass

---

### AngularJS utilities

- gulp-ng-annotate

---

## Installation

```
# Install gulp and bower as global dependency
npm install gulp bower -g
# Install boilerplate dependencies
npm install
# Start dev task
npm start
```
